Pero debo explicarles cómo surgió esta idea errónea de 
denigrar el respeto y alabar el dolor, y les daré una explicación completa del sistema, exponiendo las 
enseñanzas del gran explorador de la verdad, el artífice de la felicidad humana. Nadie rechaza, 

aborrece ni evita el respeto en sí mismo, sino las consecuencias dolorosas que conlleva para quienes no 
saben buscarlo racionalmente. Tampoco hay quien ame, 
busque o desee el dolor en sí mismo, sino las circunstancias en las que el esfuerzo y el dolor pueden
 procurarle un gran respeto. Por poner un ejemplo 
 sencillo, ¿quién se somete a un ejercicio físico 
 extenuante si no es para obtener algún beneficio? ¿Pero quién tiene derecho a criticar a quien elige 
 disfrutar de un respeto sin consecuencias negativas, o a quien evita un dolor que no produce ningún respeto 
 posterior?
